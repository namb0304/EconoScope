# EconoScope - 統計的不確実性の可視化ツール

## 概要

**EconoScope**は、教育データを題材に「統計的不確実性」を直感的に理解するためのWebアプリケーションです。

サンプルサイズの違いが推定結果の信頼性にどう影響するかを、インタラクティブな可視化を通じて学ぶことができます。

## 主な機能

### 1. シナリオ選択
3つのシナリオから分析対象を選択：
- **小テスト** (n=20) - 少人数クラス
- **中間試験** (n=100) - 1クラス分
- **学年全体** (n=300) - 学年統一テスト

### 2. 分析の問い選択
- **推定の信頼性**: この平均点の推定は、どれくらい信頼できる？
- **サンプル数の影響**: サンプル数が変わると精度はどう変わる？
- **少人数の判断**: 少人数のデータで判断しても大丈夫？

### 3. 結果の可視化
- 95%信頼区間付きの推定値表示
- サンプル数による不確実性の比較グラフ
- 教育的な解説とインサイト

## 教育的価値

このアプリケーションは以下の統計的概念を直感的に理解することを目的としています：

1. **標準誤差 (SE = σ/√n)**: サンプル数が増えると推定の精度が上がる
2. **信頼区間**: 推定値の不確実性を区間で表現
3. **サンプルサイズの重要性**: 少数データでの判断のリスク
