# EconoScope アーキテクチャ概要

## 概要
EconoScope は、学業成績データを題材に  
**「不確実性（Uncertainty）」を可視化する教育向け Web アプリケーション**です。

ベイズ推論および MCMC（マルコフ連鎖モンテカルロ）法を用いて、
サンプルサイズやデータのばらつきが結果解釈に与える影響を
直感的に理解できることを目的としています。

---

## 技術スタック

### フロントエンド
- React
- TypeScript
- Vite
- Tailwind CSS

→ 高速な開発体験と、データ可視化に向いた構成

### バックエンド
- Python
- FastAPI

→ 数値計算・API 構築のしやすさを重視

### データ
- CSV（ダミーデータ想定）
- NumPy ベースの数値計算

### デプロイ（予定）
- Railway

---

## データフロー

1. ユーザーがデータセットをアップロード、または選択
2. バックエンドで統計推論（ベイズ推論）を実行
3. 推論結果（要約統計・分布情報）を API で返却
4. フロントエンドで不確実性を含めて可視化

---

## このプロジェクトで扱う重要な概念

- 不確実性の可視化
- サンプルサイズが推論結果に与える影響
- 教育データを題材としたベイズ推論の理解
